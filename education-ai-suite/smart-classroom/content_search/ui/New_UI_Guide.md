## 🎯 文件状态系统

### 状态流转图

```
PENDING → UPLOADING → QUEUED → PROCESSING → COMPLETED
                                              ↓
                                           FAILED
```

### 状态详解

| 状态 | 图标 | 含义 | 可执行操作 |
|------|------|------|------------|
| **Pending** | ⚪ | 已选择，待上传 | 上传、删除 |
| **Uploading** | 🔄 | 上传中（显示进度条） | 取消、删除 |
| **Queued** | ⏳ | 已上传，等待处理 | 无（自动进入下一步） |
| **Processing** | ⚙️ | AI处理中（轮询状态） | 无（自动完成或失败） |
| **Completed** | ✅ | 处理完成 | 查看、删除 |
| **Failed** | ❌ | 上传或处理失败 | 重试、删除 |

## 🔘 操作按钮

### 单文件操作

| 按钮 | 含义 | 出现条件 |
|------|------|----------|
| **[↑]** | Upload - 上传此文件 | 状态为 Pending |
| **[↻]** | Retry - 重试上传 | 状态为 Failed |
| **[×]** | Remove - 从列表移除 | Pending, Completed, Failed |
| **[👁]** | View - 查看详情 | Completed（可选） |
| **[-]** | 无操作 | Queued, Processing |

### 批量操作（页面底部）

| 按钮 | 功能 |
|------|------|
| **Upload All (N)** | 上传所有待上传的文件 |
| **Clear Completed (N)** | 清除所有已完成的文件 |
| **Clear Failed (N)** | 清除所有失败的文件 |

## 📝 使用流程

### 场景 1: 上传新文件

1. **选择文件**
   - 点击 "Select Files" 按钮
   - 或直接拖拽文件到上传区域

2. **查看文件列表**
   - 文件会立即出现在表格中
   - 状态显示为 ⚪ Pending

3. **添加标签（可选）**
   - 点击文件的标签列
   - 输入标签名称（如 Math, Grade1）

4. **上传文件**
   - 点击文件行末的 [↑] 按钮
   - 或点击底部 "Upload All" 批量上传

5. **等待处理**
   - 上传时显示进度条 🔄 65%
   - 上传完成后自动开始AI处理 ⚙️
   - 每2秒自动轮询状态

6. **完成**
   - 状态变为 ✅ Completed
   - 显示处理耗时（如 "Done (52s)"）
   - 文件会保留在列表中

---

### 场景 2: 处理失败文件

1. **识别失败文件**
   - 状态显示为 ❌ Failed
   - 鼠标悬停查看完整错误信息

2. **重试上传**
   - 点击 [↻] Retry 按钮
   - 文件状态变回 Pending
   - 再次点击 [↑] 上传

3. **删除失败文件**
   - 点击 [×] Remove 按钮
   - 或使用底部 "Clear Failed" 批量清理

---

### 场景 3: 批量管理

1. **上传多个文件**
   - 选择10个文件
   - 点击 "Upload All (10)"
   - 所有文件依次开始上传

2. **清理已完成文件**
   - 5个文件处理完成
   - 点击 "Clear Completed (5)"
   - 已完成文件从列表移除

3. **查看当前状态**
   - 底部显示统计：
   - "2 pending, 3 uploading, 5 processing, 5 completed, 1 failed"

## 🐛 常见问题

### Q1: 文件上传失败怎么办？
**A**: 点击文件行的 [↻] 重试按钮，或查看错误信息排查问题。

### Q2: 如何清理已完成的文件？
**A**: 点击底部 "Clear Completed" 按钮，或单独点击每个文件的 [×] 删除。

### Q3: 可以同时上传多个文件吗？
**A**: 可以！点击 "Upload All" 批量上传，或依次点击每个文件的 [↑]。

### Q4: 上传中途可以取消吗？
**A**: 可以！在上传过程中点击 [×] 按钮即可取消。

### Q5: 处理时间很长怎么办？
**A**: 系统会自动轮询状态，无需手动刷新。大文件处理时间较长属于正常现象。

### Q6: 标签有什么用？
**A**: 标签用于后续搜索时过滤结果，建议为文件添加有意义的标签（如课程、主题等）。
